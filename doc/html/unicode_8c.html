Detailed Description

Fonctions d'affichage des cases du plateau avec des caractères unicode.

Contient deux fonctions pour l'affichage du haut d'une case et du bas d'une case en caractère unicode et avec gestion de couleur. Les caractères unicodes sont directement rentrés dans le code car sinon les lignes du plateau ne s'alignent pas correctement.
